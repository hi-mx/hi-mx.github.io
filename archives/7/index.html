原创｜如何跳过硬件限制，加入Windows11预览计划，安装预览版本？

Mingxin Lv

TPM其实就是可信平台模块，英文全称Trusted Platform Module，是一项安全密码处理器的国际标准。TPM主要利用经过安全验证的加密密钥来给设备带来更强的安全性设置，它也是许多安全应用的核心。目前TPM最新的版本是2.0版，也就是微软Windows 11操作运行所要求的版本。目前主流的电脑当中都是支持TPM2.0功能的，但大部分电脑都是默认关闭的状态，我们按win+r进入的窗口中输入【tpm.msc】进入设置页面后，可以判断出你的电脑是否开启了tpm的功能。这也就意味着如果你的电脑不支持TPM2.0的话，就无法顺利安装或升级Windows 11操作系统，很多人就卡在了这一步。
那么如何跳过硬件限制，加入Windows11预览体验计划并获取预览版本呢？我搜集到的很多方法例如修改注册表等都失效了，今天带来的方法经站长亲测有效。

温馨提示：若您对Windows 11的预览版本感兴趣，建议您通过合法渠道，加入微软的官方预览计划，以确保您获得的版本是经过官方认可和授权的，从而避免潜在的法律和安全风险。

第一步：切换至beta渠道，加入Windows 11预览计划

【工具1】OfflineInsiderEnroll

下载压缩包后解压，首先右键管理员打开OfflineInsiderEnroll，我们可以发现有“Dev渠道”“Beta渠道”和“Release Preview渠道”三个选项可以选择。其中dev渠道是目前推出的开发者预览版，而beta渠道是dev版更新完善后的测试版本，按照提示可强制切换至beta渠道，当然选dev渠道也可以。如果可以更新beta版的话，推荐直接选择更新beta版，因为有着更多的功能，更强的稳定性和更少的bug。完成这一步之后重启电脑。

打开[设置 – Windows更新 – Windows预览体验计划]，可以看到以下提示：

Device Enrolled Using OfflinelnsiderEnroll
This device has been enrolled to the Windows Insider program using OfflinelnsiderEnroll v2.6.4. If you want to change settings of the enrollment or stop receiving Windows Insider builds, please use the script. Learn more.

恭喜你已成功加入Windows 11预览计划。

第二步：获取预览版本，绕过硬件检测，完成安装

【工具2】Windows11UpdateHelper

检查更新Windows，可以看到有预览版本推送。先点击[检查更新]按钮，下载预览版本。
下载完成以后，使用Windows11UpdateHelper工具绕过安装阶段的硬件检测，等待安装即可，当然这个工具也可以在下载之前打开。打开工具之后，上方的渠道设置不用更改，只要勾选下面的监视文件夹的即可。完成以上操作后，就可以正常接收并安装Windows 11预览版本的更新了。

安装比较慢，请耐心等待，到100%安装进度时，会一段时间没有变化，这个时候一定要耐心等待安装完成。
接下来按照提示重启电脑更新，最终就成功更新到了预览版本，你可以在[设置 – 系统 – 系统信息 – Windows规格]中查看你的Windows 11最新预览版本。

第三步：开启预览版本中默认隐藏的功能

至此，你已成功完成了预览版本的更新，关于如何开启预览版本中默认隐藏的功能，请阅读本站其他文章。
